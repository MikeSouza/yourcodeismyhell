1. **_YouR_C0dE = "My Hell" or YC = "MH" for short** is a community site designed to make us all better developers/programmers/hackers.

2. **YC = "MH" is powered by OctoPress**, which you can read about below.

3. **YC = "MH" intends to be a repository of fun, insightful, and hilarious code snippets** with stories to tell. Think "TheDailyWTF" but with more collaboration.

How to submit a post:

Fork YC = "MH" on GitHub

Clone the repository: 

  git clone git@github.com:yourname/yourcodeismyhell.git

Add a post to source/_posts using Markdown or Textile formatting. View the other posts for examples.

Upload the offending code to a Gist Embed the gist into your post using:

 {{  "{% gist gist_id [filename]"  }}%}
omit [filename] for only one file.

add the post to the git repository 

  git add post_file

commit your new post 

  git commit -am "your commit message"

push your commit to your repository on github 

 git push

Submit a pull request on GitHub
