_YouR_C0dE = "My Hell" or YCMH for short is a community site designed to make us all better developers/programmers/hackers. YCMH intends to be a repository of fun, insightful and hilarious code snippets with stories to tell. Think "TheDailyWTF" but with more collaboration.

How to submit a post:

Fork YCMH on GitHub

Clone the repository: 

git clone git@github.com:yourname/yourcodeismyhell.git

Add a post to source/_posts using Markdown or Textile formatting. Name it using the YYYY-MM-DD-post-title.markdown format. View the other posts for examples.

Upload the offending code to a Gist. Embed the gist into your post using:

{{  "{% gist gist_id [filename]"  }} %}
omit [filename] for only one file.

Add the post to the git repository: 

git add "your post filename here"

Commit your new post: 

git commit -am "your commit message"

Push your commit to your repository on github: 

git push

Submit a pull request on GitHub from your forked repository.
